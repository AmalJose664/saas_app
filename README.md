# MyApp

A SaaS subscription platform built with Next.js, Supabase, Razorpay and Resend.

## Apps

- `apps/user` - Customer facing app (port 3000)
- `apps/admin` - Admin dashboard (port 3001)

## Packages

- `packages/database` - Prisma client + schema
- `packages/supabase` - Supabase client

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env.local` in each app
4. Run `npm run dev`
